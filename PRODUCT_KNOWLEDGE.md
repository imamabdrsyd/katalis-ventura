## ğŸ“ Standar Akuntansi Profesional

### Double-Entry Bookkeeping System

Katalis Ventura mengimplementasikan sistem **pembukuan berpasangan (double-entry)** yang merupakan **standar akuntansi internasional**. Setiap transaksi mempengaruhi minimal dua akun dengan prinsip:

```
Total Debit = Total Kredit
```

### Aturan Debit & Kredit

| Jenis Akun | Debit (+) | Kredit (-) | Saldo Normal |
|------------|-----------|------------|--------------|
| **ASET** | Bertambah | Berkurang | DEBIT |
| **LIABILITAS** | Berkurang | Bertambah | KREDIT |
| **EKUITAS** | Berkurang | Bertambah | KREDIT |
| **PENDAPATAN** | Berkurang | Bertambah | KREDIT |
| **BEBAN** | Bertambah | Berkurang | DEBIT |

### Contoh Pencatatan Transaksi

**Menerima Pendapatan Sewa Rp 5.000.000:**
```
Debit  : Kas/Bank           Rp 5.000.000
Kredit : Pendapatan Sewa    Rp 5.000.000
```

**Membayar Listrik Rp 500.000:**
```
Debit  : Beban Listrik      Rp 500.000
Kredit : Kas/Bank           Rp 500.000
```

### Struktur Kode Akun (Standar Indonesia)

| Range Kode | Kategori |
|------------|----------|
| **1000-1999** | Aset (Asset) |
| **2000-2999** | Liabilitas (Liability) |
| **3000-3999** | Ekuitas (Equity) |
| **4000-4999** | Pendapatan (Revenue) |
| **5000-5999** | Beban (Expense) |

### Formula Keuangan

```
Laba Kotor     = Total Pendapatan - Biaya Variabel
Laba Operasi   = Laba Kotor - Biaya Operasional
Laba Bersih    = Pendapatan - OPEX - VAR - CAPEX - TAX

ROI (%)        = (Laba Bersih / Modal Investasi) Ã— 100%

Persamaan Akuntansi:
ASET = LIABILITAS + EKUITAS
```
